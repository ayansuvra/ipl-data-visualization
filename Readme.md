# IPL Data Visualization using Python
This project analyze IPL data from kaggle using Python to discover insights about teams, players and match trends.
---
## Project Overview
 - visualized team performence, win patterns, toss decision
 - compared batting and fielding trends
 - compared win and toss decision
---
## Dataset
The data is taken from Kaggle and includes
- matches.csv -> match-level details(teams, winner, etc.)
- deliveries.csv -> contains runs, wicket, batsmen, bowlers.
---
## Tools and Libraries
- Python
- Pnadas, Numpy
- Matplotlib, Seaborn
- JupyterNotebook
